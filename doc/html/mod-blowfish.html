Blowfish module

Eggdrop uses the blowfish module to encrypt user's passwords. Please note, that when you change your encryption method later (i.e. using other modules like a md5 module) you can't use your current userfile anymore. Eggdrop can't start without an encryption module.

  1. Config file settings

1. Config file settings

Lines in bold are the lines that need to be included in the eggdrop config file if you want to use the share module.
Lines in bold italics are optional and only required if you wish to change the default behaviour for that particular feature.

Put this line into your config file to load blowfish module:

checkmodule blowfish
