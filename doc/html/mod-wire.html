Wire module

This module provides all the standard .wire commands via dcc. It's an encrypted partyline communication tool, compatible with wire.tcl>

  1. Config file settings
  2. DCC commands

1. Config file settings

Lines in bold are the lines that need to be included in the eggdrop config file if you want to use the share module.
Lines in bold italics are optional and only required if you wish to change the default behaviour for that particular feature.

Put this line into your config file to load the wire module:

loadmodule wire

2. DCC commands

.wire

Allows the user to use encrypted partyline communication
